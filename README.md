## 📌 Proyectos

### Sistema de Inventario Fastech

El proyecto consiste en un sitio web diseñado para administrar de forma eficiente los productos de la empresa, así como las operaciones de compras y ventas. Su propósito es centralizar la información y facilitar la toma de decisiones mediante un control claro y actualizado del inventario.

**Stack:** PHP · MySQL · HTML · CSS

---

### Centro Adopcion Canina

El proyecto consiste en sitio web que facilita la gestión de un centro de adopción de perros. Su objetivo es ofrecer una plataforma amigable donde los usuarios puedan conocer a los perros disponibles, iniciar procesos de adopción y consultar el historial de adopciones realizadas.

**Stack:** Python · Pandas · Scikit-Learn
